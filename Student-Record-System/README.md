# Student Record System

## Objective
Store and manage details of multiple students using an array of structures.

## Concepts Used
- Structures
- Array of Structures
- Loops
- Searching
- User Input and Output

## Student Details Stored
- Name
- Roll Number
- Semester
- Branch
- CGPA

## Features
- Store details of multiple students
- Display all student records
- Find the student with the highest CGPA

## Algorithm
1. Read the number of students.
2. Store student details using an array of structures.
3. Display all student records.
4. Compare CGPA values.
5. Display the student with the highest CGPA.

## Language
C
