# Largest Element in an Array

## Objective
Find the largest number from a list of user-entered integers.

## Concepts Used
- Arrays
- Loops
- Functions
- User Input and Output

## Algorithm
1. Read the number of elements.
2. Store the elements in an array.
3. Compare each element with the current largest value.
4. Display the largest element.

## Sample Output

Enter number of elements: 5

12
45
32
56
23

Largest Number: 56

## Language
C
