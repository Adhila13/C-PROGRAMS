# Digital Logic Calculator

## Description
A C program that simulates basic digital logic gates. The user enters two binary inputs (0 or 1) and selects a logic operation. The program then displays the corresponding output.

## Features
- AND Gate
- OR Gate
- NAND Gate
- NOR Gate
- XOR Gate
- XNOR Gate
- Input validation (accepts only 0 and 1)

## Concepts Used
- C Programming
- Conditional Statements
- Switch Case
- Logical Operators
- Digital Electronics Fundamentals

## Sample Input

Enter two inputs: 1 0

Option:
1 - AND
2 - OR
3 - NAND
4 - NOR
5 - XOR
6 - XNOR

## Sample Output

For XOR:

Output: 1

## Logic Gates Supported

| Gate | Expression |
|--------|-----------|
| AND | A && B |
| OR | A \|\| B |
| NAND | !(A && B) |
| NOR | !(A \|\| B) |
| XOR | A ^ B |
| XNOR | !(A ^ B) |
